# Guru Academy - Online Coaching Platform

A comprehensive online coaching website designed specifically for 10th grade science students, offering expert guidance, study materials, and preparation courses for CBSE board exams.

## 🎯 Features

### Core Features
- **Responsive Design**: Mobile-first approach with modern UI/UX
- **Course Management**: Comprehensive 10th class science courses with filtering
- **Study Materials**: Digital resources and practice materials
- **Student Testimonials**: Success stories and achievements
- **Contact Forms**: Easy communication with the academy
- **Mobile App Integration**: Cross-platform mobile application

### Course Categories
1. **Complete Package**
   - Complete Science Package (10th)

2. **Individual Subject Courses**
   - Physics (10th)
   - Chemistry (10th)
   - Biology (10th)
   - Mathematics (10th)

3. **Board Preparation Courses**
   - CBSE 10th Board Preparation
   - Science Crash Course (10th)

### Study Materials
- NCERT Solutions (10th)
- Question Banks (3000+ questions per subject)
- Video Lectures (300+ hours)
- Mock Tests (30+ tests)
- Lab Manuals (10th)
- Progress Tracking

## 🚀 Getting Started

### 1. Clone the repository
```
git clone <repo-url>
cd Guru_Academy
```

### 2. Setup Backend
```
cd backend
npm install
node server.js
```
- The backend will start (default: http://localhost:PORT)

### 3. Setup Frontend
- Open `frontend/index.html` in your web browser
- For admin panel, open `frontend/admin.html`

### 4. Test Responsive Design
- Check the website on different devices and screen sizes
